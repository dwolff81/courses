MPG Analysis

Expected MPG based on car characterisitcs

Data Scientist

About the app

  1. Written using the R package Shiny
  2. Leverages the mtcars dataset
  3. Intended for people who are buying a car

Benefits of using the app

  1. Helps the user select the right car in order to save on gas
  2. Ease to use
  3. The app is interactive!

How to use the app

  1. Select a car feature
  2. View the output (correlation between mpg and the selected feature)
  3. Repeat steps 1 and 2

Example output from the app

library(datasets)
mpgData <- mtcars
mpgPlot <- boxplot(mpg~am, data = mpgData)
